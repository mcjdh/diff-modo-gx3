## 🚀 Quick Start

1. **Install and run setup**:
```python
from consciousness_qwen.utils import setup_consciousness_system
setup_consciousness_system()
```

2. **Basic usage**:
```python
from consciousness_qwen import Qwen3ConsciousModel
import asyncio

async def main():
    # Create conscious model
    model = Qwen3ConsciousModel("qwen3:4b")
    model.start_consciousness()
    
    # Generate conscious response
    result = await model.conscious_generate("What is consciousness?")
    print(f"[{result['mode_used']}] {result['response']}")
    
    model.stop_consciousness()

asyncio.run(main())
```

3. **Run quick start example**:
```bash
python examples/quick_start.py
```
